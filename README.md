# FuelChain Database Project

A relational database project focused on fuel supply chain and station management.
The system was designed to model entities, transactions, and operations involved in fuel distribution using structured SQL database design principles.

---

## Overview

The project implements a database-driven system for managing fuel-related operations such as:

* fuel stations
* suppliers
* customers
* fuel transactions
* inventory/storage management
* relational data handling

The goal of the project is to demonstrate practical database modeling, normalization, and query implementation techniques.

---

## Features

* Relational database schema design
* Entity relationship modeling
* SQL query implementation
* Structured data management
* Data consistency and integrity constraints
* Backend-oriented database architecture

---

## Technologies Used

* SQL
* Relational Database Design
* SQLite 
* Python 

---

## Running the Project

1. Create the database using the provided SQL schema.
2. Execute the SQL scripts.
3. Run the application.

Example:

```bash
python main.py
```
